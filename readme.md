# Netflix Data Analysis Dashboard (Power BI)

## Overview
Developed an interactive Power BI dashboard to analyze Netflix content data, focusing on genre distribution, ratings, release trends, and country-wise availability.

## Key Features
- Interactive dashboard with filters (genre, year, content type)
- Analysis of Movies vs TV Shows distribution
- Year-wise trend analysis
- Genre and rating-based insights
- Drill-down to detailed content level

## Data Processing

- Cleaned and transformed raw Netflix dataset using SQL queries
- Performed data preprocessing including handling missing values, formatting fields, and splitting columns
- Exported cleaned data into Excel and structured it into multiple relational tables (cast, directors, countries, genres, descriptions)
- Built a relational data model to support efficient analysis in Power BI

## Insights
- Movies dominate Netflix catalog compared to TV Shows
- Significant growth in content after 2015
- Drama and International genres are most common
- TV-MA and TV-14 are dominant ratings

## Tech Stack
- Power BI
- MySQL and Excel (Data Cleaning & Preprocessing)
- Data Visualization
